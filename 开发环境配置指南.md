# 开发环境配置指南

## 一、环境要求

### 必需软件
1. **JDK 17+** - SpringBoot 3.x 要求
2. **Node.js 18+** - Vue3 开发环境
3. **Maven 3.8+** - 后端构建工具
4. **MySQL 8.0** - 数据库
5. **Redis 7.x** - 缓存服务
6. **Git** - 版本控制
7. **IDEA** - 后端开发（推荐）
8. **VS Code** - 前端开发（推荐）

### 可选软件
1. **Docker Desktop** - 容器化部署
2. **Postman** - API测试
3. **Navicat** - 数据库管理

## 二、软件安装

### 1. JDK安装
```bash
# 下载JDK 17或21
# 官网：https://adoptium.net/

# 配置环境变量（Windows）
JAVA_HOME = C:\Program Files\Java\jdk-17
PATH = %JAVA_HOME%\bin

# 验证安装
java -version
javac -version
```

### 2. Maven安装
```bash
# 下载Maven
# 官网：https://maven.apache.org/download.cgi

# 配置环境变量
MAVEN_HOME = C:\apache-maven-3.8.6
PATH = %MAVEN_HOME%\bin

# 验证安装
mvn -version

# 配置阿里云镜像（加速下载）
# 编辑 C:\Users\用户名\.m2\settings.xml
```

**settings.xml配置：**
```xml
<?xml version="1.0" encoding="UTF-8"?>
<settings xmlns="http://maven.apache.org/SETTINGS/1.0.0"
          xmlns:xsi="http://www.w3.org/2001/XMLSchema-instance"
          xsi:schemaLocation="http://maven.apache.org/SETTINGS/1.0.0
          http://maven.apache.org/xsd/settings-1.0.0.xsd">
    
    <mirrors>
        <mirror>
            <id>aliyun</id>
            <mirrorOf>central</mirrorOf>
            <name>Aliyun Maven</name>
            <url>https://maven.aliyun.com/repository/public</url>
        </mirror>
    </mirrors>
</settings>
```

### 3. Node.js安装
```bash
# 下载Node.js 18+
# 官网：https://nodejs.org/

# 验证安装
node -v
npm -v

# 配置npm镜像（加速下载）
npm config set registry https://registry.npmmirror.com

# 安装yarn（可选）
npm install -g yarn
```

### 4. MySQL安装
```bash
# 下载MySQL 8.0
# 官网：https://dev.mysql.com/downloads/mysql/

# 安装后启动服务
# Windows: 通过服务管理器启动
# 或命令行：net start mysql

# 登录测试
mysql -u root -p
```

### 5. Redis安装

```bash
cd redis-windows
```

## 三、项目初始化

### 1. 创建项目目录结构
```bash
# 在D:\vscode project下创建项目
cd "D:\vscode project"

# 克隆仓库（如果已创建）
git clone https://github.com/你的用户名/ds-system.git

# 进入项目
cd ds-system

# 创建目录结构
mkdir backend
mkdir frontend
```

### 2. 后端项目初始化（使用IDEA）
1. 打开IDEA -> New Project
2. 选择 Maven Archetype
3. 设置：
   - Name: ds-system-backend
   - Location: D:\vscode project\ds-system\backend
   - Archetype: maven-archetype-quickstart
   - GroupId: com.ds
   - ArtifactId: ds-system
4. 点击 Create

### 3. 前端项目初始化（使用命令行）
```bash
cd "D:\vscode project\ds-system\frontend"

# 使用Vite创建Vue3项目
npm create vite@latest . -- --template vue

# 安装依赖
npm install

# 安装常用依赖
npm install element-plus
npm install axios
npm install vue-router@4
npm install pinia
npm install @element-plus/icons-vue

# 启动测试
npm run dev
```

## 四、配置文件

### 后端 application.yml
```yaml
server:
  port: 8080

spring:
  datasource:
    driver-class-name: com.mysql.cj.jdbc.Driver
    url: jdbc:mysql://localhost:3306/ds_system?useSSL=false&serverTimezone=Asia/Shanghai&characterEncoding=utf8
    username: root
    password: 你的密码
  
  redis:
    host: localhost
    port: 6379
    password: 
    database: 0

mybatis-plus:
  mapper-locations: classpath:mapper/*.xml
  type-aliases-package: com.ds.entity
  configuration:
    map-underscore-to-camel-case: true
    log-impl: org.apache.ibatis.logging.stdout.StdOutImpl

# JWT配置
jwt:
  secret: ds-system-secret-key-2024
  expiration: 86400000  # 24小时
```

### 前端 vite.config.js
```javascript
import { defineConfig } from 'vite'
import vue from '@vitejs/plugin-vue'

export default defineConfig({
  plugins: [vue()],
  server: {
    port: 5173,
    proxy: {
      '/api': {
        target: 'http://localhost:8080',
        changeOrigin: true
      }
    }
  }
})
```

## 五、数据库初始化

### 创建数据库和表
```sql
-- 创建数据库
CREATE DATABASE ds_system DEFAULT CHARACTER SET utf8mb4 COLLATE utf8mb4_unicode_ci;

USE ds_system;

-- 用户表
CREATE TABLE sys_user (
    id BIGINT PRIMARY KEY AUTO_INCREMENT,
    username VARCHAR(50) UNIQUE NOT NULL,
    password VARCHAR(255) NOT NULL,
    nickname VARCHAR(50),
    email VARCHAR(100),
    phone VARCHAR(20),
    avatar VARCHAR(255),
    status TINYINT DEFAULT 1,
    create_time DATETIME DEFAULT CURRENT_TIMESTAMP,
    update_time DATETIME DEFAULT CURRENT_TIMESTAMP ON UPDATE CURRENT_TIMESTAMP
);

-- 角色表
CREATE TABLE sys_role (
    id BIGINT PRIMARY KEY AUTO_INCREMENT,
    role_name VARCHAR(50) NOT NULL,
    role_key VARCHAR(50) UNIQUE NOT NULL,
    status TINYINT DEFAULT 1,
    create_time DATETIME DEFAULT CURRENT_TIMESTAMP
);

-- 用户角色关联表
CREATE TABLE sys_user_role (
    user_id BIGINT NOT NULL,
    role_id BIGINT NOT NULL,
    PRIMARY KEY (user_id, role_id)
);

-- 商品表
CREATE TABLE product (
    id BIGINT PRIMARY KEY AUTO_INCREMENT,
    name VARCHAR(200) NOT NULL,
    code VARCHAR(50) UNIQUE NOT NULL,
    category_id BIGINT,
    price DECIMAL(10,2),
    stock INT DEFAULT 0,
    description TEXT,
    image VARCHAR(255),
    status TINYINT DEFAULT 1,
    create_time DATETIME DEFAULT CURRENT_TIMESTAMP,
    update_time DATETIME DEFAULT CURRENT_TIMESTAMP ON UPDATE CURRENT_TIMESTAMP
);

-- 商品分类表
CREATE TABLE product_category (
    id BIGINT PRIMARY KEY AUTO_INCREMENT,
    name VARCHAR(100) NOT NULL,
    parent_id BIGINT DEFAULT 0,
    sort_order INT DEFAULT 0,
    status TINYINT DEFAULT 1,
    create_time DATETIME DEFAULT CURRENT_TIMESTAMP
);

-- 订单表
CREATE TABLE orders (
    id BIGINT PRIMARY KEY AUTO_INCREMENT,
    order_no VARCHAR(50) UNIQUE NOT NULL,
    user_id BIGINT NOT NULL,
    total_amount DECIMAL(10,2),
    status TINYINT DEFAULT 0 COMMENT '0待支付 1已支付 2已发货 3已完成 4已取消',
    pay_time DATETIME,
    create_time DATETIME DEFAULT CURRENT_TIMESTAMP,
    update_time DATETIME DEFAULT CURRENT_TIMESTAMP ON UPDATE CURRENT_TIMESTAMP
);

-- 订单详情表
CREATE TABLE order_item (
    id BIGINT PRIMARY KEY AUTO_INCREMENT,
    order_id BIGINT NOT NULL,
    product_id BIGINT NOT NULL,
    product_name VARCHAR(200),
    price DECIMAL(10,2),
    quantity INT,
    create_time DATETIME DEFAULT CURRENT_TIMESTAMP

    start_time DATETIME,
    end_time DATETIME,
    status TINYINT DEFAULT 0 COMMENT '0未开始 1进行中 2已结束',
    create_time DATETIME DEFAULT CURRENT_TIMESTAMP
);
```

```yaml
  backend:
    build: ./backend
    ports:
      - "8080:8080"
    depends_on:
      - mysql
      - redis
    environment:
      - SPRING_DATASOURCE_URL=jdbc:mysql://mysql:3306/ds_system
      - SPRING_REDIS_HOST=redis

  frontend:
    build: ./frontend
    ports:
      - "80:80"
    depends_on:
      - backend

  mysql:
    image: mysql:8
    ports:
      - "3306:3306"

      - MYSQL_ROOT_PASSWORD=123456

    volumes:
```

## 八、常见问题

### 1. Maven下载慢
配置阿里云镜像（见上方Maven配置）

### 2. npm下载慢
```bash
npm config set registry https://registry.npmmirror.com
```

### 3. Redis连接失败
- 确保Redis服务已启动
- 检查端口6379是否被占用
- 检查防火墙设置

### 4. MySQL连接失败
- 确保MySQL服务已启动
- 检查用户名密码是否正确
- 检查数据库是否已创建

### 5. 端口冲突
- 后端默认8080，前端默认5173
- 如需修改，更新application.yml和vite.config.js
